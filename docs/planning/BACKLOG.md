## 이번 갱신

- 제작 패널을 닫은 뒤 controlbar의 `제작` 버튼으로 다시 열 수 있게 한다.
- 시나리오를 다시 열 때 에디터 버튼 이벤트가 이전 state에 중복 바인딩되지 않도록 정리한다.
- 교육 자료 보관함 MVP 화면을 추가한다. 샘플 교육 자료 카드, 검색, 즐겨찾기, 교육 시작 흐름을 먼저 제공한다.
- 공유 전 검사 결과를 내보내기 흐름에 연결한다. 오류가 있으면 Word/PPT/PDF 내보내기를 막고, 경고만 있으면 사용자가 확인한 뒤 계속 진행한다.
- 교육 자료 작업 폴더 내부 `.autosave/`에 자동 저장본을 남기고, 복구 가능한 자동 저장본을 보관함에서 안내한다.
- 교육 자료 상세 변경 이력을 `change-log.json`에 저장하고, 같은 `courseId` 자료를 날짜와 `contentHash` 기준으로 분류한다.
- 부서, 직군, 추가 분류, 태그 후보를 실행파일 옆 `database/classification-settings.json`에 저장하고 보관함에서 확인한다.
- 읽기 전용 실행 모드와 사번 기반 편집 권한 판정, 편집 사본 생성/취소 흐름을 추가한다.
- 패키지 필수 파일, JSON 문법, 링크 파일 누락, 압축 위험 경로, 무결성 캐시 검사를 추가한다.
