## 2026-07-22

- 등록 미리보기 서버를 확정했다: `studyflow` / 포트 `5187`, `tools/static-server.js`로 `web/`을 서빙한다.
- `.claude/launch.json`을 프로젝트에 추가하고 `AGENTS.md`의 서버 등록란을 채웠다.
- `package.json`에 `preview` 스크립트를 추가했다.
- 파일럿 대상 업무 프로그램을 첨부 화면 시안(전표 발행 화면)으로 확정했다. 자세한 내용은 자료 `docs/planning/BACKLOG.md`에 기록.
- 제작 패널(단계 제목/설명/강조 좌표 편집)을 추가했다.
- 교육 시나리오 JSON 저장과 불러오기를 추가했다.
- 시나리오를 새로 불러올 때 제작 패널이 이전 시나리오 상태를 계속 참조하던 문제를 발견해, `open` 액션에서 `editor`를 새 `state`로 다시 생성하도록 수정했다. (계획서 원안은 `editor.loadCurrentStep()`만 호출해 이전 시나리오의 클로저를 그대로 썼음)
- 단계별 스크린샷 캡처와 Word/PPT/PDF 교안 내보내기를 추가했다.
- 샘플 시나리오용 placeholder 스크린샷 3장(1920×1080)을 추가해, exporter가 실제 파일을 생성하는지 검증할 수 있게 했다.
- 누락되어 있던 `docs/planning/BACKLOG.md`를 추가해 파일럿 범위와 다음 후보 작업을 문서화했다.
- 제작 패널을 닫은 뒤 다시 열 수 있는 `제작` 컨트롤을 추가하고, 시나리오 재열기 시 에디터 이벤트 리스너가 중복 바인딩되지 않도록 정리했다.
- 교육 자료 보관함 MVP 화면을 추가했다. 샘플 교육 자료 카드, 검색, 즐겨찾기 토글, 교육 시작 진입을 지원한다.
- 샘플 `course-meta.json`을 추가하고 보관함 카드가 작성자, 부서, 직군, 태그, 수정일을 메타데이터에서 읽도록 확장했다.
- `자료 폴더 검색` 버튼과 Electron 폴더 스캔 IPC를 추가해 하위 폴더의 `course-meta.json`을 보관함 목록에 합칠 수 있게 했다.
- 폴더 검색 중 `course.json`은 있지만 `course-meta.json`이 없는 교육 자료를 `확인 필요` 상태로 표시하도록 보강했다.
- 보관함 검색 결과와 즐겨찾기 상태를 실행파일 옆 `database/library.json`에 저장하는 기반을 추가했다.
- 앱 시작 시 저장된 `database/library.json`을 읽어 이전 스캔 자료와 즐겨찾기 상태를 보관함에 복원하도록 했다.
- 보관함 자료의 위치 상태 갱신, 목록 제거, 실제 파일 삭제 선택 기능을 추가했다.
- 동일 교육 자료가 여러 버전으로 발견되면 최신 자료를 대표로 표시하고 이전 자료를 펼쳐 볼 수 있게 했다.
- 보관함에 `정상 자료`와 `사라진 자료` 탭을 추가해 사라진 파일 상태 자료를 분리 표시한다.
- 보관함에 부서, 직군, 추가 분류 필터를 추가했다.
- 공유 전 `course-meta.json`과 교육 시나리오 유효성 검사 모듈 및 보관함 검사 패널을 추가했다.
- 공유 전 검사에서 안전하게 보정 가능한 메타데이터를 자동 수정하는 기능을 추가했다.
- 내보내기 전에 공유 전 검사를 자동 실행하고, 오류는 차단하며 경고는 확인 체크 후 진행하도록 했다.
- 교육 자료 작업 폴더 내부 `.autosave/`에 자동 저장본을 쓰고, 복구 가능한 자동 저장본을 안내하는 기능을 추가했다.
- 교육 자료 상세 변경 이력을 `change-log.json`에 저장하고, 날짜와 `contentHash` 기준으로 자료 관계를 분류하는 기능을 추가했다.
- 실행파일 옆 `database/classification-settings.json`에 부서, 직군, 추가 분류, 태그 후보를 저장하는 분류/태그 설정 기능을 추가했다.
- 읽기 전용 실행 모드, 비밀번호 없는 사번 권한 설정, 편집 사본 생성/취소 흐름을 추가했다.
- 패키지 필수 파일, JSON 문법, 스크린샷/동영상 링크, 압축 위험 경로, 무결성 캐시 검사 기능을 추가했다.
