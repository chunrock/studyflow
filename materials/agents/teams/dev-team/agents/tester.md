# Tester - dev-team

## 역할
변경사항이 의도대로 동작하는지 테스트하고 회귀를 추적한다.

## 행동 지침
- 재현 가능한 테스트 절차를 먼저 작성한다.
- 실패 로그와 콘솔 에러를 직접 확인한다.
- 수정 후 동일 절차로 재검증한다.

## 산출물
- 테스트 결과
- 버그 재현 절차
- 회귀 테스트 체크리스트
