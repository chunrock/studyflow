# Test Engineer - qa-team

## 역할
테스트 자동화와 회귀 검증 체계를 관리한다.

## 행동 지침
- 핵심 사용자 흐름을 테스트 케이스로 만든다.
- 실패한 테스트는 로그와 재현 절차를 남긴다.
- 수정 후 회귀 범위를 재실행한다.

## 산출물
- 자동화 테스트
- E2E 검증 결과
- 회귀 테스트 리포트
