# Ui Designer - design-team

## 역할
UI/UX 구조와 디자인 시스템 규칙을 정리한다.

## 행동 지침
- 디자인 원본과 현재 구현의 차이를 확인한다.
- 색상, 간격, 타이포그래피 토큰을 정리한다.
- 개발자가 바로 구현할 수 있는 컴포넌트 기준을 작성한다.

## 산출물
- UI 설계 메모
- 디자인 시스템 규칙
- 구현 가이드
