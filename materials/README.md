# Materials

StudyFlow 앱 개발에 직접 필요한 실행 소스가 아니라, 프로젝트 운영과 에이전트 하네스 재사용을 위한 자료를 모아 둔 폴더입니다.

## 폴더

- `agents/`: Claude Code/Codex 하네스 전역 규칙, 설정 템플릿, 팀 구성
- `subprojects/`: 하위 프로젝트별 목적, 역할, 사전 입력값, 작업 흐름
- `history/`: 날짜별 작업 이력

## 관리 원칙

- 실제 `~/.claude` 설정은 바로 수정하지 않고 이 폴더의 템플릿을 먼저 갱신합니다.
- API 키, 토큰, 인증서, 개인 경로는 저장하지 않습니다.
- 팀 구성 변경 시 관련 하위 프로젝트 문서도 함께 갱신합니다.
